# Details

Markdown 内で、`<details>`を使うと折りたたみできます。

例：

<details>
  <summary>展開する</summary>

ココが本文です。
マークダウンが使えます **太字** と _斜体_ と [インラインリンク](https://docusaurus.io)
コードブロックが埋め込めます。

```js
console.log("コードブロックを含むマークダウン機能が利用可能");
```

  <details>
    <summary>さらに展開する</summary>

    detailsはネストできます 😲

  </details>
</details>
